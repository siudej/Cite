Features

  • Facilitates fetching formatted citations from ArXiv, MathSciNet and zbMATH.
  • BibTex and LaTex outputs are suported, with customizable styling in the latter.
  • Batch mode allows a series of references to be checked and reformatted.
  • Command line version of this tool integrates with LaTex editors and other applications (TexShop, Vim, ...).

Subscriptions and limitations

MathSciNet requires subscription, while zbMATH is limited to 3 results without subscription. However, zbMATH results can be filtered through MRef to get MathSciNet MR number and a more consistent BibTeX/LaTeX output.
